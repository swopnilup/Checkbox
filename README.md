# ✅ Checkbox — To-Do List App

A clean, minimal, and fully functional To-Do List web app built with pure **HTML**, **CSS**, and **JavaScript**. No frameworks, no dependencies — just straightforward front-end development.

---

## 🚀 Live Demo

> Open `todo-list.html` directly in your browser — no setup or installation needed.

---

## 📸 Features

### 🟡 Progress Bar
- Displays your overall task completion as a **live percentage**
- Animated progress bar updates instantly as you check off tasks
- Shows total task count and how many are done

### 📋 Task List
- Add tasks by typing and pressing **Enter** or clicking **+ Add**
- Check off tasks with a smooth animated **custom checkbox**
- Delete any task with the **× button** (appears on hover)
- Completed tasks get a strikethrough style

### 📊 Status Overview
- **Remaining** column — shows all pending tasks at a glance
- **Completed** column — shows all finished tasks
- Both columns update in **real time** as you interact

### 💾 Persistent Storage
- Tasks are saved to **localStorage** — they survive page refreshes and browser restarts
- No account or login needed

## 🖥️ How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/swopnilup/checkbox.git
   ```

2. **Open the project folder**
   ```bash
   cd checkbox
   ```

3. **Run it** — simply open `todo-list.html` in any browser
   ```bash
   start todo-list.html       # Windows
   open todo-list.html        # macOS
   xdg-open todo-list.html    # Linux
   ```

> No npm install. No build step. Just open and use.
